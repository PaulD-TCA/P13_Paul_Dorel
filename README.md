# P13_Dorel_Paul
Cette application à été créé dans le cadre du projet numéro 13 du parcours
OpenClassrooms de développeur d'application. Ce projet consiste à réaliser un
projet numérique du choix de l'étudiant.
# Information sur le projet
Ce projet appelé TEC ET ART fourni à ses utilisateurs une plate forme sur
laquelle les utilisateurs peuvent télécharger les connaissances nécessaires pour
la réalisation d’objets de toutes sortes. Ce projet souhaite avoir un une
portée sociale et se veut ouvert au plus grand nombre de personnes. L’accès aux
ressources de TEC ET ART est libre, gratuite et anonyme. Pour aller plus plus
loin et faire partie de la communauté, l’utilisateur devra créer un compte. Ce
compte permet de publier des design, de proposer des réalisations ou d’acheter les
réalisations d’un autre membre de la communauté.
# Notice d'utilisation
Ce projet étant open source tout le monde peut reproduire le programme.
Pour utiliser ce programme en local vous aurez besoin de:
- Forquer le dépôt pour le mettre sur votre dépôt local
- Créer un environnement virtuel et installer les dépendances
- Re-créer une application Django pour obtenir une SECRET_KEY Django
- Dans le dossier principal entrer "python3 manage.py runserver"
